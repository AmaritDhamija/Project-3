# Project-3

## Project Summary
This project analyzes COVID-19 by-law enforcement data to identify patterns in enforcement actions and predict ticket issuance.

## How to Run
1. Open `notebooks/analysis.ipynb`
2. Ensure dataset is in the `data/` folder
3. Run all cells from top to bottom

## Helper Module
The `src/helpers.py` file contains reusable functions for loading and cleaning the dataset.
